BeeOS v0.1

Autonomous bee breeding automation for ComputerCraft

Install in-game:
wget https://raw.githubusercontent.com/ChristopherJMiller/computercraft-beeos/main/beeos/install.lua install.lua install

# System Layers

BeeOS runs as independent, toggleable layers. Enable what you need.

ALWAYS ON

Layer 0: Passive Tracker

Scans all inventories. Builds species catalog with sample counts, template availability, drone/princess stock. Never moves items.

TOGGLEABLE

Layer 1: Apiary Manager

Monitors Industrial Apiaries. Auto-restarts dead queens. Routes products to AE2, drones to processing, princesses back to apiaries.

TOGGLEABLE

Layer 2: Sample & Templates

Maintains minimum genetic samples per species. Routes drones to Genetic Sampler. Crafts templates via crafting turtle.

TOGGLEABLE

Layer 3: Auto-Discovery

Traverses the full mutation tree via BFS. Uses the basic Mutatron to discover and catalog every reachable bee species autonomously.

# Hardware Layout

[AE2 ME System]
    | import/export buses
[Buffer Chests] --wired modem--+
                               |
[Advanced Monitor] <- [Advanced Computer] <- wired network
                               |
    +----------+-----------+---+----+----------+---------------+
[Apiaries  ] [Sampler(s)] [Imprinter] [Mutatron] [DNA Extractor]
[  1-12    ]
                                         [Crafting Turtle]

All machines connected via wired modems + networking cable. Computer orchestrates everything via pushItems/pullItems over the network.

# Features

  • Reads full bee genetics via Plethora (species, traits, active/inactive alleles)
  • Queries the entire mutation graph at runtime — no hardcoded data
  • Auto-discovers custom modpack bees (MagicBees, Gendustry, pack-specific)
  • BFS pathfinding through the mutation tree for optimal breeding order
  • Touch-screen monitor with species grid, apiary status, discovery progress
  • Toggle layers on/off independently via monitor touch or terminal commands
  • Surplus drones automatically routed to DNA Extractor
  • Crafts genetic templates via crafting turtle (no per-species AE2 patterns needed)
  • Persistent state survives computer reboots
  • Error recovery: crashed layers auto-restart without taking down the system

# Compatible Mods

CC:Tweaked Plethora Forestry Gendustry MagicBees Applied Energistics 2 Binnie Mods

Built for MeatballCraft (1.12.2) but should work with any modpack containing these mods.

# Quick Start

  1. Set up hardware: computer, monitor, wired modems, networking cable to all machines
  2. Place buffer chests with AE2 import/export buses
  3. Run the installer on the computer
  4. Run tools/scan to discover peripheral names
  5. Run tools/inspect <chest> <slot> to verify bee reading works
  6. Edit config.lua with your chest/machine names
  7. Run beeos — starts with passive tracker only
  8. Enable layers as you verify each one works
